Escopo do Produto

O produto desenvolvido tem como objetivo auxiliar os usuários da plataforma Brasil Participativo. Um dos aspectos-chave do escopo do chatbot é sua capacidade de resolver de forma eficiente as problemas comuns que os usuários possam ter. Isso inclui fornecer respostas rápidas e precisas para perguntas frequentes sobre as políticas, procedimentos e recursos disponíveis na plataforma. Ao oferecer resoluções rápidas, o chatbot tem como objetivo capacitar os usuários com as informações necessárias para se envolver de forma eficaz com a comunidade Brasil Participativo.

Requisitos

Funcionais RF01 O bot precisa interagir com vários usuários ao mesmo tempo
RF02 O bot precisa tirar dúvidas das pessoas acerca do Brasil participativo
RF03 O chatbot precisa fornecer links para páginas que o usuário deseja ir dentro do brasil participativo
RF04 O chatbot terá que auxiliar o usuário com problemas de login
RF05 O chatbot deve manter a conversa mesmo que várias abas estejam abertas, mantendo um fluxo de conversa
RF06 O chatbot deve permitir que os usuários enviem feedback sobre as conversas, se o problema foi resolvido ou não
Não Funcionais RNF01 O bot precisa estar disponível 24 horas por dia
RNF02 O acesso ao chatbot precisa ser feito através de um widget
RNF03 O chatbot terá que responder o usuário em menos de 2 minutos
RNF04 O widget do chatbot, precisa ser acessado em todas as páginas
RNF05 O chatbot deve ser desenvolvido utilizando uma arquitetura escalável para lidar com um aumento no número de usuários
RNF06 O chatbot deve funcionar nos principais navegadores: Opera, Firefox, Chrome
RNF07 O chatbot deve aderir às políticas de privacidade e segurança de dados, garantindo a proteção das informações dos usuários durante as interações
RNF08 O chatbot deve ter um fluxo de conversa intuitivo para o usuário
